## Best overall videos

1. StatQuest with Josh Starmer, `StatQuest: Principal Component Analysis (PCA), Step-by-Step`
   - Link: https://www.youtube.com/watch?v=FgakZw6K1QQ
   - Why this is the best single match for the lecture:
     - strong intuition
     - clear `2D -> 1D` geometric walkthrough
     - covers explained variance and scree plot idea
     - also connects PCA to SVD
   - Best use: the main student-facing video for this lecture.

2. StatQuest with Josh Starmer, `StatQuest: PCA - Practical Tips`
   - Link: https://www.youtube.com/watch?v=oRvgq966yZg
   - Why use it:
     - directly covers scaling
     - directly covers centering
     - directly covers how to think about the number of principal components
   - Best use: reinforcement for `why standardize` and `how to choose k`.

3. codebasics, `Machine Learning Tutorial Python - 19: Principal Component Analysis (PCA) with Python Code`
   - Link: https://www.youtube.com/watch?v=8klqIM9UvAc
   - Why use it:
     - connects theory to Python and scikit-learn
     - useful alongside the lecture's PCA code sketch
   - Best use: lab prep or post-lecture implementation practice.
